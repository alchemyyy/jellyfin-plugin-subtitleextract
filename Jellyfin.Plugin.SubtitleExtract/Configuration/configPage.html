Override Expressions

Boolean expressions using C# syntax that override the normal filters above. Available variables: LANGUAGE (e.g. "eng", "jpn", "und"), TYPE (e.g. "subrip", "ass", "PGSSUB"), TITLE (the subtitle track title). You can use standard C# string methods and operators: ==, !=, &&, ||, !, .Contains(), .StartsWith(), .EndsWith().
When enabled, subtitles matching this expression are always extracted, bypassing all other filters.
When enabled, subtitles matching this expression are always skipped, regardless of all other filters. Takes precedence over accept override.
